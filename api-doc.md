# UpSkill server API

## Endpoints

List of available endpoints:

- `POST /users/register`
- `POST /users/login`
- `GET /external/val/agents`
- `GET /external/val/agents/:uuid`
- `GET /pros/`

access_token needed for routes below:

- `POST /coaching/:id`
- `GET /coaching/`
- `PATCH /coaching/:id`
- `PUT /coaching/:id`
